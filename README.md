# Modelo multivariado del precio del oro

Entregable académico para la materia Economía de Minas.

## Información

- Universidad Nacional de Colombia
- Facultad de Minas
- Autores: Alejandro Ramírez y Paola Patiño
- Profesor: Jheyson Bedoya

## Visualización

El archivo principal para GitHub Pages es `index.html`, que redirige al entregable ubicado en `dashboard/index.html`.

## Contenido

- Portada académica.
- Introducción, objetivos, marco teórico, metodología y ecuaciones.
- Resultados del modelo LightGBM + Optuna.
- Validación temporal y diagnóstico fuera de muestra.
- Pronósticos para 2026.
- Simulación Monte Carlo hasta 2035.
- Discusión, limitaciones y conclusiones.

## Regeneración

Para regenerar el HTML desde los resultados:

```bash
python codigo/generar_dashboard.py
```
